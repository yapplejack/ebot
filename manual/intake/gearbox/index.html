UltraPlanetary Assembly

Tools required:

  • 3/32 allen wrench
  • 5/64 allen wrench


Parts required:

  • Qty(1) x NEO 550 motor
  • Qty(1) x UltraPlanetary 550 Motor Pinion Gear (comes in pack of two)
  • Qty(1) x UltraPlanetary 5mm Hex Output
  • Qty(1) x UltraPlanetary 550 Motor Plate
  • Qty(1) x UltraPlanetary 5:1 Cartridge
  • Qty(1) x UltraPlanetary 1/2” Hex Adaptor
  • Qty(1) x UltraPlanetary Hardware Pack




Begin assembling the UltraPlanetary gearbox by pressing the UltraPlanetary 550 Motor Pinion onto the NEO 550’s output shaft using an arbor press or similar. A video guide showing how to do this can be found here. REV recommends securing the pinion with Loctite 635 or 638 (“Green” Loctite) or equivalent.

Take the UltraPlanetary 550 Motor Plate and place the flat side against the face of the NEO 550. Line up the two larger mounting holes with the NEO 550 mounting holes and screw it in place with two M3 8mm button head screws.


Line up the input of the UltraPlanetary 5:1 Cartridge with the motor’s pinion and push the cartridge into place, twisting it until the splines and alignment features line up and sit flush.



Use five M3x10mm to secure the ½” hex output to the output stage of the Ultraplanetary.


Take the UltraPlanetary 5mm Hex Output and line it up with the output of the 5:1 cartridge. The output of the Output can be held in place or rotated to make this easier.


Bolt the Hex Output cartridge, 5:1 gear cartridge, and Motor Plate together using six M3 20mm socket head screws. Make sure not to overtighten these screws, they should be finger tight.
